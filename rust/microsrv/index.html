Lightweight Rust Microservices Framework

发表于 2016-04-06 更新于 2017-11-01 分类于 Programming
本文字数： 7.1k 阅读时长 ≈ 6 分钟

轻量级Rust微服务框架介绍与小结. Hyper Crate在Rust开发中比较常见, 特别是Rust微服务开发中, 本文将介绍以下几种轻量级的Rust微服务Crates:

  • Rouille
  • Nickle
  • Rocket
  • Gotham

其中, Rouille、Nickel和Rocket均为同步框架, Gotham为异步微服务框架, 常见的异步微服务Crates有:

  • Hyper
  • Actix/Actix-web
  • Gotham (内部使用Hyper和Tokio俩大Crate进行开发)
